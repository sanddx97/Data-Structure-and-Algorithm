Stack 

A data structure in which all insertions and deletion are made at one end, called the top of the Stack

LIFO data structure: last in, first out

Push(item): push item to the top of the Stack

Pop(item): remove and return the top item

Peek(item): return the top item without removing item

Is_empty(item): return true if the stack is is empty

Applications of the stack

1. Reverse polish for evaluating arithmetic expressions
2. Syntax parsing
3. Cold stack: space for parameters and local variables is created
4. Used in recursion
5. Undo and redo operations in word processors
6. Low level assembly programming
